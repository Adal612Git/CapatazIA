Convenciones transversales

  • La API oficial del MVP vive bajo /api/v1.
  • IDs del dominio en UUID v4.
  • Fechas en ISO 8601 UTC.
  • Éxitos con data + meta; errores con error + traceId.
  • Mutaciones críticas aceptan Idempotency-Key.
